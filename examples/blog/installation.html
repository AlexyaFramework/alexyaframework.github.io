First of all download the template and Alexya.
Extract the template the root directory of your web server and extract alexya in a directory upper to the root directory of your web server.
For example, if the root directory of your web server is /var/www/public_html extract the template to /var/www/pubic_html/ and alexya to /var/www/alexya/.

Now that we have installed all files we need to set up the .htaccess and index.php files.
The .htaccess will redirect all request to the index.php file, that way the router will handle SEO URLs. This is how it looks:

RewriteEngine On

RewriteCond $1 !\.(gif|jpe?g|png|css|js)$ [NC]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d

RewriteRule ^(.*)$ index.php/$1 [L,QSA]

Now the index.php file, it will only include the index file in alexya directory:
<?php
/**
 * Include actual index.php
 */
require_once("../alexya/index.php");

And that's all, there's no need of more, now we can start designing the database.
