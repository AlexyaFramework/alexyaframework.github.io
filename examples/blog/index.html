Here we will make a CMS for a blog using all Alexya components.
As I'm not a good web designer we will use Bootstrap's blog template which I find kinda cool and clean.

First thing we're going to do is to install Alexya, instead of having everything in same directory we're going to keep back-end files separated from front-end files, this will be the directory structure:

+---public_html
|    |    +---css
|    |    +---js
|    |    +---img
|    |    +---templates_c
+---alexya
|    +---Alexya
|    |   (Alexya's core)
|    +---Application
|    |   +---ORM
|    |   +---Module
|    |   |   +---Controller
|    |   |   +---Model
|    |   |   +---Presenter
|    |   |   +---View
|    |   +---Page
|    |       +---Controller
|    |       +---Model
|    |       +---Presenter
|    |       +---View
|    +---config
|    +---logs
|    +---packages
|    +---translations
|    +---vendor

Next thing we're going to do is designing the database so we have something to work with. I always say that the project should fit the database and not the database should fit the project.
