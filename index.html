Alexya Framework

The intelligent Loli Framework

Alexya framework is a lightweigh PHP 7 framework that supports HMVC, MVVM and MVC design patterns that can be mixed together to form the ultimate HMV(VM)C design pattern.


It also provides helpers for database access + query generation + ORM, DOM generation, socket client, input validation and many more features
