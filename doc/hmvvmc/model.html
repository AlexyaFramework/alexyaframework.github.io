The model represents the M in MV(VM)C and it contains the data that will be used to print the view. The class \Alexya\Model is the base class for all models of your application.

Method summary:

Name Parameters Return value Description
onInstance void This method is called when the model is instanced, instead of overriding the constructor you can override this method.
Properties summary:
Name Type Visibility Description
request \Alexya\Http\Request Public The request that was sent to instance the model.
triad \Alexya\Triad Public A reference to the \Alexya\Triad object that instanced the model.

Model's function is fair eay as it only contains the data that the View will display: 

<?php
namespace Application\Model;

use \Alexya\Model;

/**
 * Model for the 'User' page
 *
 * @author Manulaiko <manulaiko@gmail.com>
 *
 * @package \Application\Model
 */
class User extends Model
{
    /**
     * User name
     */
    public $name = "";

    /**
     * User email
     */
    public $email = "";

    /**
     * User password
     */
    public $password = "";

    /**
     * Set data
     */
    public function onInstance()
    {
        $this->name     = "Test";
        $this->email    = "test@test.test";
        $this->password = "test";
    }
}
