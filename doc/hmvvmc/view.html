The view represents the V in MV(VM)C and it contains the HTML to display. The class \Alexya\View is the class that handles all views of your application. You don't need to extend this class since most of the views are only a file with HTML that will be sent as the body response object returned by the controller.

Method summary:

Name Parameters Return value Description
setGlobal string $key: Key in $_data array.
mixed $value: Value for $key 		void Adds a variable to the array that can be accessed by all views. This is a static method.
render string Renders the view and returns its output.
__toString() string Alias for render.
Properties summary:
Name Type Visibility Description
_globalData Array Protected Contains the variables that will passed to the renderer and can be accessed to ALL views. This is a static property.
_path string Protected Path to the file to render.
_isDir boolean Protected Whether $_path is a directory or not
triad \Alexya\Triad Public A reference to the \Alexya\Triad object that instanced the view.

The view can be either a Smarty template (.tpl), a PHP script (.php) or a static HTML file (.html). Depending of the extension of the file will be rendered as any of it.
The variables that has been assigned to the view can be accessed as an ordinary PHP variable. The values of the $_data array (defined in \Alexya\Tools\Object) overrides the values of the $_globalData array.
Example of a Smarty template (file name: Application/View/User.tpl):

<html>
<head>
    <title>User</title>
</head>
<body>
    <table>
    <tr>
        <td>Name</td>
        <td>{$name}</td>
    </tr>
    <tr>
        <td>Password</td>
        <td>{$password}</td>>
    </tr>
    <tr>
        <td>Email</td>
        <td>{$email}</td>
    </tr>
    </table>
</body>
</html>
Example of a PHP script (file name: Application/View/User.php):
<html>
<head>
    <title>User</title>
</head>
<body>
    <table>
    <tr>
        <td>Name</td>
        <td><?php echo $name; ?></td>
    </tr>
    <tr>
        <td>Password</td>
        <td><?php echo $password; ?></td>>
    </tr>
    <tr>
        <td>Email</td>
        <td><?php echo $email; ?></td>
    </tr>
    </table>
</body>
</html>
Example of a HTML static page (file name: Application/View/User.html):
<html>
<head>
    <title>User</title>
</head>
<body>
    <table>
    <tr>
        <td>Name</td>
        <td>Test</td>
    </tr>
    <tr>
        <td>Password</td>
        <td>test</td>>
    </tr>
    <tr>
        <td>Email</td>
        <td>test@test.test</td>
    </tr>
    </table>
</body>
</html>
