Alexya lets you generate HTML and CSS easily with the classes located under the package \Alexya\DOM.

HTML

The class \Alexya\DOM\Tag provides an easy way to generate HTML code with a OOP interface in PHP. You can extend this class to add more tags.

Method summary:

Name Parameters Return value Description
addChild array|Tag $tag: Tag (or tags) to add. void Adds a child tag.
The parameter can be an object of type Tag or an array containing numerous objects of type Tag.
parseAttributes string Parses and returns tag's attributes.
fetch string Renders the tag and all its children tags.
render void Renders the tag and all its children tags and prints the result.
addClass string $class: Class to add. void Adds a class to the tag.
removeClass string $class: Class to remove. void Removes a class from the tag.
__set string $attribute: Attribute name.
string|array $value: Attribute value(s). 		void Adds an attribute to the tag.
__get string $attribute: Attribute name. string|array Returns an attribute from the tag.
Properties summary:
Name Type Visibility Description
_childs array Protected Contains all children tags.
_name string Protected Tag name.
_attributes array Protected Tag attributes.
_isSelfClosing boolean Protected Wether the tag is self closing or not.
innerHTML string Public Tag's inner HTML.

The constructor accepts as parameter the tag name as a string and an associative array containing attributes.
To render the tag and all its child tags use the method \Alexya\DOM\Tag::render
To add a child to this tag use the method \Alexya\DOM\Tag::addChild
To add raw HTML to the tag change the variable innerHTML
Example:

$html = new \Alexya\DOM\Tag("html", [
    "lang" => "en"
]);
$head = new \Alexya\DOM\Tag("head");
$body = new \Alexya\DOM\Tag("body");

$title = new \Alexya\DOM\Tag("title");
$title->innerHTML = "Test";

$p = new \Alexya\DOM\Tag("p");
$p->innerHTML = "Test paragraph";

$head->addChild($title);
$body->addChild($p);
$html->addChild([
    $head,
    $body
]);

$html->render();
/**
 * Output:
 * <html>
 *     <head>
 *         <title>Test</title>
 *     </head>
 *     <body>
 *         <p>Test paragraph</p>
 *     </body>
 * </html>
 */

A part from the \Alexya\DOM\Tag there are other classes for HTML generating:

  • \Alexya\DOM\Form
  • \Alexya\DOM\Image
  • \Alexya\DOM\Input
  • \Alexya\DOM\Link
  • \Alexya\DOM\List
Example:
$form = new \Alexya\DOM\Form([
    "name"   => "test_form",
    "action" => "/Test",
    "method" => "post",
    "class"  => [
        "form",
        "form-wrapper"
    ]
]);

$form->addChild(new \Alexya\DOM\Input([
    "name"  => "username",
    "type"  => "text",
    "class" => "username_input"
]));
$form->addChild(new \Alexya\DOM\Input([
    "name"  => "password",
    "type"  => "password",
    "class" => "password_input"
]));

$form->addChild(new \Alexya\DOM\Input([
    "type"  => "submit",
    "value" => "Login"
]));

$form->render();

/**
 * Output
 *
 * <form name="test_form" action="/Test" method="post" class="form form-wrapper">
 *     <input name="username" type="text" class="username_input" />
 *     <input name="password" type="password" class="password_input" />
 *     <input type="submit" value="Login" />
 * </form>
 */

$image = new \Alexya\DOM\Image([
    "src" => "#",
    "alt" => "alt_title"
]);

$image->render();

/**
 * Output
 *
 * <img src="#" alt="alt_title" />
 */

$link = new \Alexya\DOM\Link([
    "href" => "#"
]);
$link->innerHTML = "Click me";

$link->render();

/**
 * Output
 *
 * <a href="#">Click me</a>
 */

$list = new \Alexya\DOM\List(\Alexya\DOM\List::ORDENED_LIST);

for($i = 0; $i < 5; $i++) {
    $list->addItem(new \Alexya\DOM\Link([
        "href" => "#"
    ]));
}

$list->render();

/**
 * Output
 *
 * <ol>
 *     <li><a href="#"></a></li>
 *     <li><a href="#"></a></li>
 *     <li><a href="#"></a></li>
 *     <li><a href="#"></a></li>
 *     <li><a href="#"></a></li>
 * </ol>
 */

CSS

The class \Alexya\DOM\CSS provides an OOP way for generating CSS stylesheets.

Method summary:

Name Parameters Return value Description
addRule string $selector: Rule selector.
array $content: Rule content. 		void Adds a CSS rule.
deleteRule string $selector: Rule selector void Removes a CSS rule.
getRule string $selector: Rule selector array Returns a rule, throws a \Alexya\Exceptions\NotFound if $selector does not exist in the array.
fetch string Renders the CSS.
render void Renders the CSS and prints the result.
Properties summary:
Name Type Visibility Description
_rules array Private Contains all CSS rules.

The method \Alexya\DOM\CSS::addRule lets you add a new rule, the first parameter is the CSS selector and the second parameter is an array that contains the CSS.

The method \Alexya\DOM\CSS::deleteRule lets you remove a rule, the parameter is the CSS selector.

The method \Alexya\DOM\CSS::getRule returns an array containing the CSS, the parameter is the selector of the rule.
The method \Alexya\DOM\CSS::render renders the CSS file.
Example:

$css = new \Alexya\DOM\CSS();

$css->addRule("*", [
    "margin"  => "0px",
    "padding" => "0px"
]) ;
$css->addRule("h1", [
    "text-decoration" => "underline",
    "color"           => "#3c3c3c"
]);

$css->render();
//Output:
// * {
//     margin: 0px;
//     padding: 0px;
// }
//
// h1 {
//     text-decoration: "underline",;
//     color: #3c3c3c;
// }
