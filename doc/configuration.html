Alexya's configuration files are located under the directory config.
Each file of this directory is a PHP script that returns an associative array with the configuration parameters.

  • The file alexya.php contains the configuration values for all components.
  • The file application.php contains the configuration values for your application.
  • The file database.php contains the configuration values for the database connection, they're explained here.
  • The file paths.php contains the path related constants definition.
  • The file routes.php contains the routes used by the router, they're explained here.

Application's configuration file (application.php) returns an empty array by default since it should contain the configuration values for your application.

Paths configuration file (paths.php) contains the constants definition related to paths, all path constants ends with the directory separator value.
Defined constants:
