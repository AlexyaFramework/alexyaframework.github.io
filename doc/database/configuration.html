The file config/database.php contains the parameters that will be used for connecting to the database, make sure they fit your requirements.

Key Value type Default value Description
enable boolean true Wether we're going to use the \Alexya\Database\Connection class or not.
host string localhost IP or hostname of the database server.
port int 3306 Port on which is running the server.
type string mysql Server type, at the moment Alexya supports the servers listed on PDO's documentation.
username string root Server authentication username.
password string Server authentication password.
database string alexya Name of the database to connect.
options Array 
[]
 Alternative PDO options.
