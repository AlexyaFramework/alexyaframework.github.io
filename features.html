Alexya Framework

The intelligent Loli Framework

The ultimate HMV(VM)C design pattern

Alexya not only brings you the posibility to use any of HMVC, MVVM and MVC design patterns, it also offers the posibility of mixing all the design patters to make the ultimate HMV(VM)C design pattern.

Easy to use and consistent syntax

Alexya is really easy to use, you just need to download it and start building your application with a consistent syntax.

Fully configurable and extensible

Every single feature can be configured to fit your needs and you can install 3rd party packages for extra fun effect.

Easy database interaction

Alexya provides classes for connecting to databases, generating queries and ORM helpers.

Socket client

Alexya contains SocksWork for an easy way to connect to a server and sending/reading packets.

Localization support

Alexya also provides classes for localized output like text translation and date/number formatting.

Showcase

As easy to use as this

Application/Controller/Test.php

<?php
namespace Application\Controller;

use \Alexya\Controller;
use \Alexya\Http\Response;

class Test extends Controller
{
    public function index() : Response
    {
        $response = new Response();
        $response->body = "<h1>Alexya Framework!</h1>"
    }
}

Output of http://localhost/Test

<h1>Alexya Framework!</h1>
